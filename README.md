# Debian Optimizer Script
A script to optimize debian server and manage some services.

## 🚀 Quick Start

Simply run the one-line command below.

🌍 For Global Users:
```
apt update && apt install -y curl && curl -sLo ~/deb_optimizer.sh https://raw.githubusercontent.com/G3arB0xX/debian-optimizer-script/main/deb_optimizer.sh && bash ~/deb_optimizer.sh
```

🇨🇳 For Mainland China Users:
```
apt update && apt install -y curl && curl -sLo ~/deb_optimizer.sh https://ghfast.top/https://raw.githubusercontent.com/G3arB0xX/debian-optimizer-script/main/deb_optimizer.sh && bash ~/deb_optimizer.sh
```
